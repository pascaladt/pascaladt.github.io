Description

TIntegerDoubleListAdt is essentially TIntegerListAdt with support for bidirectional iterators. Its two new methods are implemented by means of casting the result of ForwardStart (ForwardFinish) to TIntegerBidirectionalIterator. This means that the class which inherits from TIntegerDoubleListAdt must return an iterator derived from TIntegerBidirectionalIterator in those two functions (although their return types are marked as TIntegerForwardIterator)..
