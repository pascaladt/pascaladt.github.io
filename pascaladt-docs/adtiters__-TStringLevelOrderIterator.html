Description

TStringLevelOrderIterator is derived from TStringTreeTraversalIterator and indirectly from TStringBidirectionalIterator instead of directly from TStringForwardIterator in order to provide it with an interface common with other traversal iterators. However, it's highly inefficient to use its Retreat method - it takes an O(n) time (one step)..
