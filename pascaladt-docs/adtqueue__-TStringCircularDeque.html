Description

Implements a 0-based circular double-ended queue. A circular deque (double-ended queue) uses one array to store its items. The array is treated as circular, e.g. if there is not enough place at the front to push an item, the sequence of items 'wraps around' and the item is stored at the back of the allocated place. Requires reallocation only when the whole preallocated space becomes used-up..
