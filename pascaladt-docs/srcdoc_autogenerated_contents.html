Modules

adt23tree

This unit provides an implementation of a 2-3 tree - a data structure that provides all set operations with the worst-case O(log(n)) time and can be used as a concatenable priority queue..

adtalgs

This unit provides various algorithms to be used with the PascalAdt containers.

adtarray

This unit provides an array (TArray) and a wrapper around delphi dynamic arrays (TPascalArray)..

adtavltree

This unit provides the AVL-tree (TAvlTree).

adtbinomqueue

This unit provides the binomial heap priority queues (TBinomialQueue)..

adtbintree

This unit provides a general binary tree - TBinaryTree..

adtbstree

This unit provides a binary search tree (TBinarySearchTree).

adtcont

This unit provides abstract base classes (interfaces) for many different types of containers.

adtcontbase

This unit provides the base class for all containers..

adtdarray

This unit handles low-level memory management via TDynamicArray.

adtexcept

This unit provides several exception classes used by the library..

adtfunct

This unit provides abstract functor interfaces as well as some commonly used concrete functors.

adthash

This unit provides implementations of a hash table (THashTable) and of a scatter table (TScatterTable).

adthashfunct

This unit provides some fairly good hashing functions for a number of data types.

adtiters

All abstract iterator classes are defined in this file, as well as some basic routines dealing with them..

adtlist

This unit provides three implementations of lists: TSingleList, TDoubleList and TXorList..

adtlog

This unit provides logging facilities.

adtmap

This unit provides map wrappers around set containers..

adtmem

This unit provides facilities for memory-management..

adtopers

This unit overloads operators for common operations on iterators.

adtqueue

This unit provides two implementations of double-ended queues - TCircularDeque and TSegDeque.

adtsegarray

This unit implements a dynamic low-level segmented array (TSegArray).

adtsplaytree

This unit provides an implementation of a splay-tree.

adtstralgs

This unit provides various string algorithms, including the Knuth-Morris-Pratt, Boyer-Moore and Karp-Miller-Rosenberg algorithms..

adttree

This unit provides general (in the sense of 'unconstrained') tree class - TTree..

adtutils

This unit presents a few basic, simple utilities like Min, Max, Exchange, ...
