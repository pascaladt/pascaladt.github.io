Description

A splay tree. Guarantees amortized O(log(n)) complexity for all set operations. This data structure is very fast when a small number of all items is accessed most of the time, which is usually the case. However, if you know that the items are going to be distributed randomly, it would be better to use another structure with the worst-case O(log(n)) time, such as the AVL-tree, or the red-black tree..
