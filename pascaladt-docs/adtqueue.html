adtqueue

Uses in interface

adtmem, adtfunct, adtcontbase, adtiters, adtcont, adtdarray, adtsegarray

Description

This unit provides two implementations of double-ended queues - TCircularDeque and TSegDeque. There is also a type TDeque present, which is equivalent to TSegDeque - the best implementation of a double-ended queue for most uses.
