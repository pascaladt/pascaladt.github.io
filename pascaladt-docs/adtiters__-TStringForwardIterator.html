function Delete(finish : TStringForwardIterator) : SizeType; overload; virtual;

Visibility

public

Source

adtiters.pas on line 856

Description

Deletes all items from the range [self, finish). After the deletion the iterator points to finish. Invalidates all other iterators (in general). Returns the number of items deleted. After the deletion the iterator points to finish. The default implementation uses only abstract methods and traverses the range twice. This may be a bit inefficient, so this is re-implemented by some containers.

Complexity

O(n) invocations of Delete();.
