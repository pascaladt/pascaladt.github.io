Description

A pair of two set iterators. Used frequently in connection with sets. An object of this class is owned by the container which is the owner of Start and is destroyed automatically (so you do not have to bother). However, Start and Finish are not owned by the object (as they are already owned by their 'parent' containers).

See also

TIntegerForwardIteratorRange, TMapIteratorRange
