Description

TStringDoubleListAdt is essentially TStringListAdt with support for bidirectional iterators. Its two new methods are implemented by means of casting the result of ForwardStart (ForwardFinish) to TStringBidirectionalIterator. This means that the class which inherits from TStringDoubleListAdt must return an iterator derived from TStringBidirectionalIterator in those two functions (although their return types are marked as TStringForwardIterator)..
