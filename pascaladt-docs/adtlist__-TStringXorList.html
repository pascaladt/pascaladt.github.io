Description

Implements a XOR list. A XOR list is essentially a doubly-linked list, but it uses a dirty trick of xor'ing the next and previous node pointers together, thanks to which it uses the same amount of memory as a singly-linked list. Generally, TStringDoubleList should be used, and TStringXorList only in critical code where low memory usage is vital..
