Description

KmrFindSubstrings

Declaration

function KmrFindSubstrings(const str : String; start, finish : IndexType; len : SizeType) : TCardinalArray;

Visibility

global

Source

adtstralgs.pas on line 289

Description

Implements the Karp-Miller-Rosenberg algorithm for computing the equivalence classes of all substrings with length len in str. If the table returned contains the same number at index i and j, it means that the substrings of length len starting at i and at j are equal. Only positions from [start,finish-len) in the returned array contain valid indices !! Others are undefined. Len is the desired length of substrings, should be > 1. Start and finish designate the range in which the search is performed.

Complexity

O(n*log(n)).
