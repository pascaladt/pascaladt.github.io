Description

This unit provides implementations of a hash table (THashTable) and of a scatter table (TScatterTable). Scatter tables are sometimes called closed hash tables, the first container metioned is sometimes also called an open hash table or a chained hash table. THashTable Uses lists to resolve collisions, whereas TScatterTable uses pseudo-random probing technique and keeps all items in one array. They both perform all set operations in average O(1) time and worst-case O(n). However, for most uses THashTable is recommended as it is slightly faster (better constant factors). On the other hand, TScatterTable consumes less memory..
