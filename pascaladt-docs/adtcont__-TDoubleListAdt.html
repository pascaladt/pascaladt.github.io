Description

TDoubleListAdt is essentially TListAdt with support for bidirectional iterators. Its two new methods are implemented by means of casting the result of ForwardStart (ForwardFinish) to TBidirectionalIterator. This means that the class which inherits from TDoubleListAdt must return an iterator derived from TBidirectionalIterator in those two functions (although their return types are marked as TForwardIterator)..
