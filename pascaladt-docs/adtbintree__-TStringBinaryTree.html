TStringBinaryTree.ExtractNodePreOrder

Declaration

function ExtractNodePreOrder(var node : PStringBinaryTreeNode; fadvance : Boolean) : PStringBinaryTreeNode;

Visibility

public

Source

adtbintree.pas on line 868

Description

Disconnects node from the tree so that the pre-order of other nodes is preserved. Returns the parent of the node actually disposed (this is not necessarily node^.Parent), nil if this was the last node. Node is disposed (but not the item). After extraction node points to the next node according to pre-order. Fadvance indicates whether to assign node the next node in pre-order. This is because advancing to the next node may take an O(h) time.

Complexity

O(h), where h is the height of the tree.

function ExtractNodeInOrder(var node : PStringBinaryTreeNode; fadvance : Boolean) : PStringBinaryTreeNode;

Visibility

public

Source

adtbintree.pas on line 889

Description

Disconnects node from the tree so that the in-order of other nodes is preserved. Returns the parent of the node actually disposed (this is not necessarily node^.Parent since items may be swapped inside the tree), nil if this was the last node. node Is disposed (but not the item). After the extraction node points to the next node according to the in-order (if fadvance is true). Fadvance indicates whether to assign node the next node in in-order. This is because advancing to the next node may take O(h) time.

Complexity

O(h), where h is the height of the tree.
