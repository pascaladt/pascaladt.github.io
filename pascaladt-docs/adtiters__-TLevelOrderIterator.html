Description

TLevelOrderIterator is derived from TTreeTraversalIterator and indirectly from TBidirectionalIterator instead of directly from TForwardIterator in order to provide it with an interface common with other traversal iterators. However, it's highly inefficient to use its Retreat method - it takes an O(n) time (one step)..
