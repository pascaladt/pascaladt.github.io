adtavltree

Uses in interface

adtmem, adtfunct, adtcontbase, adtiters, adtcont, adtbintree, adtbstree

Description

This unit provides the AVL-tree (TAvlTree). The AVL-tree is a data structure allowing all set operations to be performed in worst-case O(log(n)) time and it keeps items sorted..
