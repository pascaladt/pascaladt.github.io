adtbstree

Uses in interface

adtiters, adtcont, adtcontbase, adtfunct, adtbintree, adtmem

Description

This unit provides a binary search tree (TBinarySearchTree). It also provides a binary search tree base class (TBinarySearchTreeBase) used to implement other data structures basing on a binary search tree..
