Description

Represents a sequential list of items supporting forward iterators. The requirement that the pointers stored in the container be pairwise unequal does not have to be satisifed for classes descended from this one. (And you, therefore, have to re-implement the TIntegerForwardIterator.Delete(TIntegerForwardIterator) method);.
