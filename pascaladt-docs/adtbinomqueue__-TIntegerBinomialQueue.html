Description

Implements a binomial priority queue. A binomial priority queue is a data structure allowing to perform all concatenable priority queue operations in O(log(n)). In contrast to an ordinary heap, where the Merge operation takes O(n) time, in a binomial queue it may be performed in O(log(n)) time. On the other hand, the First operation takes O(log(n)) time instead of O(1) like in an ordinary heap. However, you amy always pre-fetch and cache the item if you find it necessary;.
